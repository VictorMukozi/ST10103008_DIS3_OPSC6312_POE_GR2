Bird Zone
=========

Bird Zone is an innovative mobile application designed to simplify the process of finding and booking birdwatching locations. Focused primarily on South African hotspots, the app provides enthusiasts with a seamless way to discover, explore, and plan their birdwatching adventures.

Purpose
-------
The purpose of Bird Zone is to:
- Help users easily locate birdwatching areas near them.
- Provide essential details such as distance, pricing, operating hours, and routes.
- Enable seamless reservations with confirmation emails.
- Support educational content, bird species guides, and rare bird alerts.
- Allow users to log and view past birdwatching observations.

Design Considerations
---------------------
- User-Friendly Interface: Simple navigation from login through booking.
- Secure Authentication: Registration and login with stored user credentials.
- Customisation: Toggle between metric/imperial units, set maximum travel distance.
- Interactive Mapping: Displays user’s current location, hotspots, and directions.
- Data Recording: Users can save observations and review them across devices.
- Scalability & Reliability: Designed for smooth performance and growth.

Features
--------
- Log-In & Registration – Secure user authentication.
- Main Hub – Explore hotspots through image buttons and search functionality.
- Interactive Map – Locate birdwatching areas within a set radius.
- Booking System – Reservation forms with email confirmation.
- Educational Content – Field guides, tutorials, and conservation articles.
- Rare Bird Alerts – Notifications when uncommon species are spotted nearby.
- Observation Recording – Log and review personal birdwatching history.

Technology Stack
----------------
- Frontend: Mobile-first interface (frameworks to be confirmed).
- Backend: Secure database for authentication, settings, and observations.
- Maps Integration: Interactive mapping APIs.
- Version Control: GitHub for source code management.
